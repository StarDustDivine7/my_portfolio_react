# Project Images

Add your project screenshots here with these filenames:

- `fintech-app.png` - FinTech Mobile App screenshot
- `healthcare-app.png` - Healthcare Management System screenshot
- `social-app.png` - Social Media Platform screenshot
- `erp-app.png` - ERP Solution screenshot
- `ai-assistant.png` - AI Query Assistant screenshot
- `video-call-app.png` - Video Calling App screenshot

## Recommended Specifications

- **Format**: PNG or WebP
- **Dimensions**: 1200x800px (3:2 aspect ratio)
- **File Size**: < 500KB (optimized for web)
- **Content**: Show the main interface or key features of your project

## Tips

- Use high-quality screenshots that showcase your work
- Ensure images are well-lit and clear
- Consider adding device mockups for mobile apps
- Compress images before uploading (use tools like TinyPNG)

## Temporary Placeholders

Until you add your images, the portfolio will show placeholder backgrounds with project names.
